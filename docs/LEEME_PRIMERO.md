# Léeme Primero

Para usar el radar en otra PC:

1. Cargar el firmware de `firmware/Proyecto_RADAR_ultras_nico.ino` en el Arduino.
2. Conectar el Arduino por USB.
3. Ejecutar `RadarUltrasonicoWeb.exe`.
4. Elegir el puerto COM desde la interfaz.
5. Presionar `CONECTAR`.

Si no aparece el puerto COM, revisar el driver USB de la placa Arduino.
